## Run Locally

Clone the project

```bash
  git clone https://github.com/aeioyu/thonburi-covid19.git
```

Go to the project directory

```bash
  cd thonburi-covid19
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
